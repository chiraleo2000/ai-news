Hidden Thoughts Are Not Secret: Reasoning Trace Exposure in LLMs

งานวิจัยนี้เปิดเผยช่องโหว่ด้านความปลอดภัยร้ายแรงใน LLM ที่ใช้ chain-of-thought หรือ extended reasoning โดยค้นพบว่า internal reasoning traces ของโมเดลสามารถถูก extract ออกมาได้โดยผู้ไม่ประสงค์ดี แม้ developer จะไม่ได้ตั้งใจเปิดเผย ช่องโหว่เกิดจากการที่ reasoning step ถูก embedded ในรูปแบบที่สามารถ infer ได้จาก output distribution รวมถึง timing attacks และ output probability probing นักวิจัยสาธิตการโจมตีบน GPT-o series และ Claude Extended Thinking ที่สามารถดึง system prompt fragments และ private reasoning ออกมาได้ ผลการวิจัยนี้มีนัยสำคัญสำหรับบริษัทที่ใช้ reasoning model ในระบบที่มีข้อมูลความลับ และชี้ให้เห็นว่า 'hidden' reasoning ไม่ได้ปลอดภัยอย่างที่คิด ช่องโหว่ reasoning trace exposure เป็น zero-day class ใหม่สำหรับ LLM security ที่กระทบ production systems ที่ใช้ extended thinking หรือ CoT models อยู่ทันที ตรวจสอบว่า reasoning traces ของ LLM ที่ใช้อยู่สามารถ leak ผ่าน output probing ได้หรือไม่ เพิ่ม output sanitization และ rate limiting สำหรับ API calls ที่ใช้ extended thinking models
