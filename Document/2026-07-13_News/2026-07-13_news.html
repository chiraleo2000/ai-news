Game Theory Driven Multi-Agent Framework Mitigates Language Model Hallucination

Impact: แนวทาง proactive memory agent ช่วยแก้ปัญหา behavioral state decay ใน agent ที่ทำงานระยะยาว ซึ่งเป็นอุปสรรคสำคัญต่อความน่าเชื่อถือของ AI agent ในงานที่ซับซ้อนและใช้เวลานาน
