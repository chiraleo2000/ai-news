# Anthropic เปิดตัว Claude Tag ให้ทีมแท็ก @Claude เป็นเพื่อนร่วมงานบน Slack

Anthropic เปิดตัว Claude Tag วิธีใหม่ในการผสาน AI เข้ากับการทำงานเป็นทีมบนแพลตฟอร์ม Slack โดยผู้ใช้งานสามารถแท็ก @Claude ใน channel เพื่อมอบหมายงานได้เหมือนเพื่อนร่วมงานจริง ๆ ฟีเจอร์นี้ทำงานในรูปแบบ multiplayer ที่ทีมเห็นและรับช่วงงานต่อจากกันได้ พร้อม ambient mode ที่ Claude จะอัปเดตข้อมูลสำคัญเชิงรุกโดยอัตโนมัติ ภายใน Anthropic เองนั้น 65% ของโค้ดจากทีมผลิตภัณฑ์ถูกสร้างโดย Claude Tag เวอร์ชันภายใน Claude Tag ทำงานบนโมเดล Opus 4.8 มีระบบ RBAC ควบคุมการเข้าถึงข้อมูลอย่างเข้มงวด เปิดให้ลูกค้า Claude Enterprise และ Team ใช้งานแบบ beta แล้ววันนี้ และจะมาแทนที่แอป Claude in Slack เดิมโดยสมบูรณ์ภายใน 30 วัน

## Tech Insights

**Impact**: Claude Tag เปลี่ยนโมเดลการทำงานกับ AI จาก one-on-one chat เป็น multiplayer team collaboration

**Developer Actions**:
- ทดลองใช้ Claude Tag beta บน Slack ขององค์กรโดยเร็ว เพื่อสำรวจการ integrate @Claude เข้ากับ workflow ของทีม engineering
- ศึกษา Agent Identity and Access Model ของ Anthropic เพื่อออกแบบ permission structure ที่ปลอดภัยก่อน deploy ใน production

**Business Actions**:
- ประเมิน use case ที่ Claude Tag จะลด overhead ของทีม เช่น งาน research, drafting, หรือ code review
- ขอ launch credit จาก Anthropic สำหรับลูกค้า Enterprise/Team เพื่อเริ่ม trial
