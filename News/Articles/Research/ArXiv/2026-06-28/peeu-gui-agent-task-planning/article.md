# PEEU: การเสริมพลัง GUI Agent ด้วยการสำรวจประสบการณ์อัตโนมัติ

**แหล่งที่มา:** [arXiv:2606.27330](https://arxiv.org/abs/2606.27330) | ACL 2026 Main | 25 มิ.ย. 2026

## สรุปสั้น

งานวิจัยที่ได้รับการยอมรับใน ACL 2026 นำเสนอวิธี PEEU ที่ช่วยให้ GUI Agent ขนาด 7B พารามิเตอร์สามารถวางแผนงานเว็บได้ดีกว่าโมเดล Qwen2.5-VL-32B ขนาดใหญ่กว่า 4 เท่า

## สรุปงานวิจัย

ทีมวิจัยนำเสนอวิธี PEEU (Planning Experience Exploration and Utilization) สำหรับ GUI Agent Multimodal ที่ทำงานบนเว็บ PEEU ทำงานด้วยการสำรวจสภาพแวดล้อมอัตโนมัติเพื่อค้นหาประสบการณ์ และใช้ Hindsight Experience เพื่อสังเคราะห์ข้อมูลฝึกระดับสูงที่สอดคล้อง ผลการทดลองบน Benchmark จริงแสดงว่าโมเดล 7B ของ PEEU ทำได้ 30.6% ซึ่งดีกว่า Qwen2.5-VL-32B ที่ใหญ่กว่า 4 เท่าอย่างมีนัยสำคัญ ยืนยันว่าข้อมูลฝึกที่ออกแบบดีสำคัญกว่าขนาดโมเดล

## ผลกระทบ

PEEU ชี้ให้เห็นว่าโมเดล MLLM ขนาดเล็ก 7B สามารถเอาชนะโมเดล 32B ในงาน GUI Task Planning ด้วยข้อมูลฝึกที่ออกแบบดี เปิดโอกาสสำหรับ AI Agent ที่ทำงานบน Edge และรักษาความเป็นส่วนตัวได้
