## Body

OpenAI released 'Codex-Maxxing for Long-Running Work', a whitepaper by Jason Liu focused on advanced Codex usage for complex, multi-session projects. The guide addresses a core enterprise challenge: how to use AI coding agents for work spanning days or weeks, not just single prompt-response cycles.

Key strategies covered:
- Using Codex as a persistent workspace that preserves project context across sessions
- Breaking ambitious goals into verifiable, checkpointable steps  
- Managing complex multi-stream workflows with multiple concurrent Codex agents
- Knowing when to delegate to Codex vs. when human review is essential
- Building documentation and structured outputs that Codex can reference in future sessions

The guide is available as a PDF at cdn.openai.com/pdf/codex-maxxing and complements the broader Codex ecosystem expansion (Codex Security, Daybreak) announced the same day.
