## สรุปข่าว

Cohere ประกาศเปิดตัว North Mini Code ซึ่งเป็น coding model รุ่นแรกของบริษัทที่ออกแบบมาสำหรับ developer โดยเฉพาะ โมเดลนี้เน้นความสามารถในงาน practical software engineering มากกว่า benchmark score โดยทำงานในลักษณะ agentic coding เช่น การเขียน code ตาม requirements การ debug และการทำ code review การเปิดตัวนี้ถือเป็นก้าวสำคัญของ Cohere ในการขยายจากตลาด enterprise NLP สู่ coding tools market ซึ่งกำลังแข่งขันสูง North Mini Code ยังปรากฏใน HuggingFace community blog สะท้อนให้เห็นการสร้าง community presence ของ Cohere ในโลก open source
